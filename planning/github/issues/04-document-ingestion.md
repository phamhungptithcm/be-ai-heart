# Issue 04: Document Ingestion and Classification

## Title

Expand document ingestion into a first-class heart capability

## Labels

- `type:feature`
- `type:docs`
- `priority:p0`
- `track:heart-core`

## Milestone

`M1 Heart Core MVP`

## Objective

Make project documents part of the durable memory layer, not optional side context.

## Scope

- markdown and text ingestion hardening
- classification quality
- configurable document roots
- secure exclusion behavior
- preparation for pdf/docx support

## Acceptance Criteria

- business, requirements, technical, and execution docs are classified reliably
- context packs return relevant document references
- ignored documents stay out of retrieval
- roadmap for pdf/docx extraction is documented

## Dependencies

- Issue 01

## Out of Scope

- OCR and scanned-image extraction
